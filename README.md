# Saajit Irfahn — Personal Portfolio Website

A personal portfolio website for Saajit Irfahn, Analytics Lead and Data Engineer based in London. The site showcases professional experience, education, projects, and skills in a clean, animated single-page layout.

Live site: **https://saajit1997.github.io/saajit_profile/**

---

## What the Page Is About

This is a single-page personal portfolio designed to act as a professional online presence. It covers:

- **Hero** — Introduction with profile photo, name, job title, and links to contact and LinkedIn
- **About** — Professional background summary with key achievement stats (savings generated, hours automated, degrees held)
- **Experience** — Full work history including Analytics Lead at Amelco UK and Graduate Process Engineer at Worley Parsons
- **Education** — MSc Business Analytics (Distinction) from UCL and MEng Chemical Engineering from the University of Nottingham
- **Projects** — Smart Contract Entity Extraction and Reddit Data Scraper & Sentiment Analysis
- **Skills** — Full tech stack organised by category (languages, data, cloud, databases, visualisation)
- **Contact** — Email, phone, LinkedIn and GitHub links

## What Was Built

### Stack
- Pure **HTML5** and **CSS3** — no frameworks or build tools required
- Vanilla **JavaScript** for all animations and interactivity
- Hosted via **GitHub Pages** (free, no server needed)

### Features
- **Typewriter effect** — name and job title type out on page load
- **Floating particle canvas** — Chelsea blue and gold particles drift across the hero background
- **Scroll-triggered animations** — cards and sections fade up as the user scrolls
- **Count-up stats** — numbers animate from zero when they come into view
- **Floating profile photo** — square-framed profile image with a blue-to-gold gradient border and gentle float animation
- **Active nav highlighting** — navigation links highlight in gold as the user scrolls through sections
- **Fully responsive** — layout adapts cleanly for mobile and tablet screens

### Design
- Inspired by [megpatakota.com](https://megpatakota.com/)
- Colour theme drawn from **Chelsea FC** — deep navy background (`#06101f`), royal blue (`#034694`), and gold accents (`#dba111`)
- Font: [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts

## How to Run Locally

No installation needed. Simply open `index.html` in any browser:

```
file:///C:/Users/<your-username>/path/to/saajit_profile/index.html
```

Or clone the repo and open the file:

```bash
git clone https://github.com/saajit1997/saajit_profile.git
cd saajit_profile
open index.html
```
